PiRingo Python Software

alloff.py	Turns all the LEDs OFF
allon.py	Turns all the LEDs ON
flashdemo.py	Runs through a sequence of flashing/chasing sequences (similar to Kickstarter video)
swing.py	Uses the switches to move a single LED left/right depending which switch is pressed
swing2.py	As swing.py but lights all LEDs in the direction of movement. Winner is the one who gets to the other side!
